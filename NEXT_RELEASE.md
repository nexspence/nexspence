## v1.14.0

### ✨ Features

- **Native installers** — Linux `.deb`/`.rpm`, macOS & Windows archives built by GoReleaser, with systemd / launchd / Windows-service integration. The binary now embeds the web UI (self-contained, no separate frontend needed). New [docs/install-local.md](docs/install-local.md) covers native setup plus reverse-proxy (nginx/Caddy) and multi-node load-balancer configs. Requires external PostgreSQL.
